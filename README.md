## Overview

fz-connect is an HTML-based monitoring bot built for FiveM setups that need a clear, live view of server activity. It highlights the current connected-player total and state updates in a format that is easy to place inside dashboards, admin pages, or embedded widgets.

This version is intended for server owners and administrators who want a light, readable status display that fits into existing FiveM workflows. The emphasis is on live updates, straightforward integration, and presentation that can be reused across different layouts without adding extra overhead.

## FAQ

**How do I update the monitor?**  
Swap in the newer project files, then recheck your configuration values before restarting the page or widget.

**Where do I change the server being monitored?**  
Look in the main HTML file or the related settings file used by the interface, then edit the server connection details there.

**Why is the player count not showing?**  
Make sure the FiveM status source is reachable, the server details are correct, and the page has been refreshed after your changes.

**Can I use this inside a dashboard or widget?**  
Yes. The interface is meant for dashboard-style display and works in web panels or server widgets.

**What should I do if the layout looks wrong?**  
Check the HTML structure, styling, and embed container dimensions so the monitor fits the space where it is shown.
